# NekoTranslate Browser Extension

Translate any manga or comic in place and in real time.

## Build it yourself

- Have Docker installed on a Linux machine, cd into this repository
- `$ ./01-build.sh`
- `$ ./02-run.sh`
- `$ npm install`
- For Chrome: `$ npm run build`
- For Firefox: `$ npm run build:firefox`
- `$ exit`
- The path of the extension ZIP file is `./dist/dist.zip`

## FAQ

A: It's basically the internal codename of NekoTranslate.
